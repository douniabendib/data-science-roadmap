# 💸 Apply Discount Function (Beginner)

A Python lab that calculates the final price of an item after applying a percentage discount.

---

## 📂 File
- `apply_discount.py` → defines the function and validates inputs.

---

## 🧑‍💻 User Stories
- Define a function named `apply_discount` with two parameters: `price` and `discount`.  
- If `price` is not a number → return `"The price should be a number"`.  
- If `discount` is not a number → return `"The discount should be a number"`.  
- If `price <= 0` → return `"The price should be greater than 0"`.  
- If `discount < 0 or discount > 100` → return `"The discount should be between 0 and 100"`.  
- Otherwise, calculate the discount and return the final price.

---
