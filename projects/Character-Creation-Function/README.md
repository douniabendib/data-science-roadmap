# 🎮 Character Creation Function

A Python function that generates a character with three stats:  
- **Strength (STR)**  
- **Intelligence (INT)**  
- **Charisma (CHA)**  

Stats are displayed using filled dots (`●`) and empty dots (`○`) on a scale of 10.

---

## 📂 Files
- `character.py` → contains the `create_character` function.

---

## ⚙️ Rules
- **Name**: must be a string, ≤ 10 characters, no spaces.  
- **Stats**: must be integers between 1 and 4.  
- **Total points**: must equal **7** across STR, INT, and CHA.  

---

## ▶️ How to Run

```bash
python character.py
