# 🌦️ Build a Travel Weather Planner (Beginner)

A Python lab that uses conditional statements to decide if commuting is possible based on distance, weather, and available transport.

---

## 📂 File
- `travel_weather_planner.py` → defines travel variables and prints commuting eligibility.

---

## 🧑‍💻 User Stories
Your program must satisfy the following requirements:

- `distance_mi`: a number representing the distance to travel in miles  
- `is_raining`: a boolean representing if it is currently raining  
- `has_bike`: a boolean representing if the user has a bicycle  
- `has_car`: a boolean representing if the user has a car  
- `has_ride_share_app`: a boolean representing if the user has a ride‑share app

---
